# MAJESTÉ MARKET

African professional marketplace PWA for Benin.

## 🌐 Live Site
https://majeste-presse-pro.netlify.app

## 📋 Description
Full-featured marketplace PWA connecting clients with 
local professionals across multiple service categories 
in Benin, West Africa.

## ⚙️ Features
- Multi-category professional listings
- Firebase real-time database
- Single Page Application (SPA) with hash routing
- Installable PWA with offline support
- Toast notification system
- Schema.org structured data for SEO
- WhatsApp contact integration
- Luxury dark/gold UI design

## 🛠️ Technologies
- HTML, CSS, JavaScript
- Tailwind CSS
- Firebase (Firestore)
- PWA / Service Worker
- Netlify (deployment)
